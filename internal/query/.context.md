# internal/query

## Purpose

Prometheus-compatible HTTP API for querying stored metrics. Embeds the Prometheus PromQL engine for full query language support.

## Functionality

- Instant query: `GET/POST /api/v1/query`
- Range query: `GET/POST /api/v1/query_range`
- Series lookup: `GET/POST /api/v1/series`
- Label names: `GET/POST /api/v1/labels`
- Label values: `GET /api/v1/label/<name>/values`
- Metadata: `GET /api/v1/metadata`
- Build info: `GET /api/v1/status/buildinfo`
- PromQL engine integration via Prometheus library
- Prometheus JSON response format (status, data, errorType, error, warnings)
- Timestamp parsing (RFC3339 and Unix seconds)
