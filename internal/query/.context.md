# internal/query

## Purpose

Prometheus-compatible HTTP API for querying stored metrics. Embeds the Prometheus PromQL engine for full query language support.

## Functionality

- Prometheus HTTP API endpoints (`/api/v1/query`, `/api/v1/query_range`, etc.)
- PromQL engine integration via Prometheus library
- Metadata endpoints (series, labels, label values, metadata)
- JSON response formatting in Prometheus API format

## Files

| File | Description |
|------|-------------|
| `api.go` | Prometheus HTTP API route handlers |
| `engine.go` | PromQL engine initialization and query execution |
| `interfaces.go` | Interfaces consumed by this package |
