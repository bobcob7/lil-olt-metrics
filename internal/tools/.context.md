# tools

## Purpose

Tracks Go development tool dependencies via a `tools.go` file with a `//go:build tools` tag. This ensures tool versions are pinned in `go.mod` for reproducible builds without including them in the compiled binary.

## Functionality

- Pins gofumpt, moq, and golangci-lint versions in go.mod
- Tools are installed via `GOBIN=$(GOBIN) go install` in the Makefile

## Files

| File | Description |
|------|-------------|
| `tools.go` | Build-tagged import file that pins tool versions in go.mod |
