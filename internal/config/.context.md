# internal/config

## Purpose

Configuration loading, defaults, and validation for lil-olt-metrics. Supports YAML files with environment variable overrides.

## Functionality

- Load config from YAML file with `Load(path string) (*Config, error)`
- Apply environment variable overrides (LOM_ prefix, underscore-separated nesting)
- Apply sensible defaults for all settings (zero-config startup)
- Validate config values at load time
- Custom Duration and ByteSize types for YAML parsing

## Files

| File | Description |
|------|-------------|
| `config.go` | Config struct definitions, Load function, defaults, env overrides, validation |
| `config_test.go` | Unit tests for defaults, YAML, env overrides, precedence, validation |
