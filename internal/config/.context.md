# internal/config

## Purpose

Configuration loading, defaults, and validation for lil-olt-metrics. Supports YAML files with environment variable overrides.

## Functionality

- Load config from YAML file
- Apply environment variable overrides (LOM_ prefix)
- Apply sensible defaults for all settings
- Validate config values

## Files

| File | Description |
|------|-------------|
| `config.go` | Config struct definitions, loading, defaults, validation |
