# internal/ingest

## Purpose

Handles OTLP metrics ingestion over gRPC and HTTP, translating the OTLP metric model into Prometheus-compatible time series for storage.

## Functionality

- OTLP-to-Prometheus metric translation (Gauge, Sum, Histogram)
- Resource attribute mapping (configurable label map + promote list)
- Scope name/version as `otel_scope_name`/`otel_scope_version` labels
- Metric name sanitization for Prometheus compatibility
- Unit suffix and type suffix handling
- Exemplar extraction with trace/span ID preservation

## Files

| File | Description |
|------|-------------|
| `translator.go` | OTLP-to-Prometheus translator, writes to store.Appender |
| `translator_test.go` | Unit tests for each metric type, labels, sanitization |
