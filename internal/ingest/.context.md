# internal/ingest

## Purpose

Handles OTLP metrics ingestion over gRPC and HTTP, translating the OTLP metric model into Prometheus-compatible time series for storage.

## Functionality

- OTLP gRPC server implementing `MetricsService/Export`
- OTLP HTTP handler for `POST /v1/metrics`
- OTLP-to-Prometheus metric model translation
- Resource attribute extraction and label mapping
- Metric name sanitization
- gzip decompression

## Files

| File | Description |
|------|-------------|
| `grpc.go` | OTLP gRPC server implementation |
| `http.go` | OTLP HTTP handler implementation |
| `translator.go` | OTLP metric model to Prometheus time series conversion |
| `interfaces.go` | Interfaces consumed by this package |
