# Integration Tests

## Purpose

End-to-end integration tests that exercise the full system: OTLP ingestion through translation, storage, and Prometheus query API response. Tests run against a real in-process server with ephemeral ports.

## Functionality

- Full server harness (gRPC ingest, HTTP ingest, Prometheus query API on ephemeral ports)
- End-to-end ingestion-to-query validation for all metric types
- FS persistence verification via WAL replay
- Concurrent load testing with race detection
- Error handling validation

## Files

| File | Description |
|------|-------------|
| `integration_test.go` | All integration tests with `//go:build integration` tag |

## Running

```bash
# Run all integration tests
go test -tags integration ./internal/integration/

# With race detection
go test -tags integration -race ./internal/integration/

# Verbose
go test -tags integration -v ./internal/integration/
```
