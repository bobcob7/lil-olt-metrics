# Plans

## Purpose

Implementation roadmap for lil-olt-metrics. Each plan file is a self-contained unit of work with clear dependencies, scope, and acceptance criteria. Plans are ordered so that each can be implemented independently once its dependencies are met.

## Milestone Map

- **After Plan 08**: Working MVP — ingest OTLP, query via PromQL, in-memory only
- **After Plan 09**: Durable storage with crash recovery
- **After Plan 11**: Full OTLP feature coverage
- **After Plan 13**: Finished product

## Dependency Graph

```
01 Foundation
├── 02 Config
├── 03 Core Types
│   ├── 04 Memory Store
│   │   ├── 06 Ingestion ←── 05 Translator ←── 03
│   │   └── 07 Query API ←── 03
│   │       └── 08 Server Wiring ←── 02, 06, 07
│   │           └── 09 FS Persistence ←── 04
│   └── 10 Remote Backends ←── 02
├── 11 Extended OTLP ←── 05, 06
├── 12 Integration Tests ←── all
└── 13 Documentation ←── all
```

Plans 09, 10, and 11 can be implemented in parallel once their dependencies are met.
