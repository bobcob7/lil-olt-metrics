# lil-olt-metrics

## Purpose

A minimal, single-process metrics server that ingests OTLP metrics and serves them via the Prometheus HTTP API. Designed for edge deployments, dev environments, and lightweight production use.

## Functionality

- OTLP gRPC and HTTP ingestion (protobuf and JSON)
- Prometheus-compatible HTTP API for querying (works with Grafana)
- Pluggable storage: built-in FS engine, Prometheus remote, or VictoriaMetrics remote
- OTLP-to-Prometheus metric model translation
- Zero-config startup with sensible defaults
