# cmd/server

## Purpose

Application entrypoint. Wires all components together via constructor injection and manages the server lifecycle (startup, signal handling, graceful shutdown).

## Functionality

- Config loading
- Component initialization and dependency wiring
- Signal handling (SIGINT/SIGTERM)
- Graceful shutdown orchestration

## Files

| File | Description |
|------|-------------|
| `main.go` | Entrypoint — loads config, wires components, starts servers |
