# cmd/server

## Purpose

Application entrypoint. Wires all components together via constructor injection and manages the server lifecycle (startup, signal handling, graceful shutdown).

## Functionality

- CLI flags: `--config` for config file path, `--version` for version info
- Config loading via `config.Load()`
- Logger initialization (level/format from config)
- Pluggable store backend: memory (default), fs (WAL + compaction), prometheus, victoriametrics
- Translator, OTLP handlers, PromQL engine, query API construction
- Three listeners: gRPC (4317), OTLP HTTP (4318), query API (9090)
- Health check endpoint (`/healthz`) on query API port
- Build-time version injection via ldflags
- Graceful shutdown on SIGINT/SIGTERM with 30s drain timeout

## Files

| File | Description |
|------|-------------|
| `main.go` | Entrypoint — loads config, wires components, starts servers, handles shutdown |
